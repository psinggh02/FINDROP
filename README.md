## Findrop

Findrop is a collection of vue and vuetify components that can be plugged into any web application with minimal change.

## List of Components

Camera
DatePicker
Drawer
IdleTimeout
Modal
Navebar
Snackbar
Steps

## Usage

Inline:  `code`

``` JavaScript
In package.json

'findrop': '1.2.0'

import findrop from 'findrop';

let component_name = findrop.<component_name>({
    ...options
});
```

``` HTML
<div>{{modal}}</div>
```
